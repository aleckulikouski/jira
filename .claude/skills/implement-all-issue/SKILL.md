# Locate the ISSUES

Issues are located in the `issues/` folder. File naming pattern is `issues/NNN-short-title.md` (e.g. `issues/001-add-user-auth.md`). Use /implement-issue to implement each issue one by one starting with the lowest numbered issue. Order of implementation should be from the lowest numbered issue to the highest numbered issue.
